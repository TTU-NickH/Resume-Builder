# CodePup AI Resume Builder 🐶💻

A full-stack resume builder that allows users to create, improve, and export professional resumes using AI.

---

## 🚀 Features

- Add jobs and skills
- AI-powered resume bullet improvement (Gemini API)
- Generate formatted resume in real-time
- Export resume as a PDF
- Desktop app using Electron

---

## 🛠 Tech Stack

- **Frontend:** HTML, Bootstrap, Tailwind
- **Backend:** Node.js, Express
- **Database:** SQLite
- **AI:** Google Gemini API
- **Desktop App:** Electron

---

## 📦 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/YOURUSERNAME/Resume-Builder.git
cd Resume-Builder

### 2. Install Dependencies
cd backend
npm install

### 3. Start the backend server
node server.js
  -Should see "Resume Builder Backend running on http://localhost:8000"

### 4. Launch the Electron App
Open a second terminal and run these commands.
cd backend
npm run electron
